# decision-support-system-bucharest-city-hall

# Project Overview
This project presents the development of a Decision Support System (DSS) designed to support data-driven decision-making within Bucharest City Hall.
The system integrates structured public-sector data and analytical models to improve transparency, efficiency, and strategic planning.

- Organizational structure and human resources data

# Methodology
- Relational database design and implementation (Microsoft Access)
- Data cleaning, validation, and preprocessing (Microsoft Excel)
- Descriptive statistical analysis and data analysis
- Development of interactive dashboards in Power BI
- Short-term revenue forecasting using time-series methods
- Multi-criteria decision analysis using the TOPSIS method with interval data

# Tools Used
- Microsoft Access
- Microsoft Excel
- Power BI

# Key contributions
- Designed and implemented a relational database integrating budget, contracts, public procurements, organizational structure, and human resources data
- Performed data cleaning, validation, and consistency checks using Excel
- Conducted descriptive statistical analysis and data analysis with Pivot Tables and Charts
- Built an interactive Power BI dashboard for monitoring key performance indicators
- Developed short-term revenue forecasts using multiple time-series methods
- Solved a multi-criteria decision-making problem for prioritizing public investments
